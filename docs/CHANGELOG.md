# FairFamily — 변경 이력 (Changelog)

이 문서는 작업 세션에서 적용한 주요 변경사항을 정리합니다.

## v1.0.2 — UI/UX 다듬기 + 가이드 투어 + 다국어 강화

### 🐛 출시 차단 이슈 수정 (이전)
- Google Play "Broken Functionality" 반려 해결: `AndroidManifest.xml`의 `MainActivity` 패키지 경로(`com.fairfamily.app.MainActivity`) 수정으로 실행 시 `ClassNotFoundException` 크래시 제거.
- 네이티브 **Google 로그인** 동작화: WebView에서 `signInWithPopup`이 안 되는 문제를 `@capacitor-firebase/authentication`(`signInWithGoogle` → `signInWithCredential`)로 해결.
- **카메라/갤러리** 선택 동작화 (`@capacitor/camera`, `CameraSource.Prompt`).

### 🗂️ 통합 카테고리 시스템
- 대시보드 레이더·로그·아이 방을 **8개 카테고리**로 통일: 시간/돈/선물/학교/1대1/마음/즐거움/건강.
- "한번에 적기" — 자유 문장을 Claude AI가 카테고리별로 자동 분류.
- 빈 칸 저장 허용 + 누락 항목 추천.

### ⏱️ 입력 항목
- 출생연도 → 나이 자동 계산.
- **횟수(times)** 기록 (N번 표시).
- **함께한 시간(hours)** — "시간" 카테고리일 때 금액 대신 시간 입력(별도 필드, 금액 합계와 분리).
- 가운데 팝업 달력(월·연도 이동).
- 아이 방 입력 폼에도 8개 카테고리 + 나이 + 날짜(오늘/다른 날) + 시간 추가.

### 📊 대시보드
- **나이별 비교**: 지표 버튼(돈/생일선물/시간/1on1/학부모참석/크리스마스/옷값/교육비)별 비교, 각 버튼 고유 색 + 글라스모피즘.
- 비교 표 자체의 "더보기" 제거 → "전체" 셀에서 이벤트 3개 초과 시 셀별 더보기.
- 공정성 점수(Fair %) = 아이들 간 상대적 균형 지표.

### 📈 라이프 그래프
- viewBox 너비를 실제 컨테이너 픽셀에 맞춰 **가로 full-width + 중앙 정렬** (점·글자 크기 고정).
- 세로 길이 확대, 연결선 강조.
- "now" 라벨은 앱에서만 크게(20), 웹은 15.
- 이벤트 목록은 상위 3개만 표시.

### 🧭 가이드 투어
- 대시보드 "둘러보기" 버튼 → 실제 카드/메뉴 위에 **스포트라이트 + 말풍선**으로 단계별 안내(9단계).

### 💬 도움말(FAQ)
- 콘텐츠를 언어 파일로 전역 분리, 언어 설정에 따라 한/영 표시, 닫기(X) 버튼 추가.
- 사용법 안내를 화살표 대신 **원형 숫자 단계**로 표시.

### 🤖 AI 조언
- 언어 설정에 따라 **AI 인사이트/답변이 한국어 또는 영어**로 생성(Claude 프롬프트에 언어 지시 추가).

### 🎨 디자인/가독성
- 라이트 테마 전반, 작은/연한 글자 가독성 개선, 타이포 위계(13→14→15→18→20→30+).
- 제목 글꼴: 한국어 `Bagel Fat One`, 영어 `Climate Crisis`.
- 하단 네비 활성 표시 강화, 설정 화면(데이터 삭제/사용법/카테고리 설명) 추가.

### 📱 네이티브 앱
- WebView가 데스크톱보다 작게 렌더되는 문제 → 네이티브에서 전체 UI `zoom: 1.18`.
